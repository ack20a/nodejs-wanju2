## 一键部署 nodejs 到 Heroku  [![Deploy](https://www.herokucdn.com/deploy/button.png)](https://heroku.com/deploy)

使用方法：上传index.js,package.json,start.sh三个文件，确保启动文件名称是index.js,启动即可

特点:需要端口，有网页，有订阅

缺点:不支持hy2,因为端口用来订阅了，如果需要hy2可以使用index2.js
